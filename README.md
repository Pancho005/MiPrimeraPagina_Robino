# Titulo

Listado:
- 1 activar el entorno virtual desde terminal: . .venv/Scripts/activate
- 2 activar la pagina desde terminal: python manage.py runserver
- 3 ingresar a la pagina http://127.0.0.1:8000/ donde va a iniciar nuestro proyecto desde la pagina inicio
- 4 para agregar un libro se lo puede hacer desde la pestaña crear libro
- 5 una vez que agregemos un libro nos va a llevar a la lista de libros guardados
- 6 tambien se puede agregar autor y genero desde sus pestañas respectivas
