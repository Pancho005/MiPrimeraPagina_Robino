# Titulo

Listado:
- 1 activar el entorno virtual desde terminal: . .venv/Scripts/activate
- 2 activar la pagina desde terminal: python manage.py runserver
- 3 ingresar a la pagina http://127.0.0.1:8000/
- 4 para agregar un libro se lo puede hacer desde http://127.0.0.1:8000/agregar-libro/titulo/autor donde titulo poner el nombre del libro y en autor el nombre del autor del libro
- 5 listar los libros agregados desde http://127.0.0.1:8000/lista-libros/
